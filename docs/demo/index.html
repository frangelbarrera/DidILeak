Want to scan your own export?

Install DidILeak locally — it takes 30 seconds: 

pip install didileak

# scan your ChatGPT export
didileak report ~/Downloads/chatgpt-export/conversations.json --outdir ./reports

# open the dashboard
open reports/didileak_report.html
